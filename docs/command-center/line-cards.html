ACP + Crypto in LINE Vision & Integration Points

        

          
How ACP / x402 Flow Into LINE

          
    
            
  • Agent Commerce Handoff — AI agent (ChatGPT) creates ACP checkout → notify buyer via LINE Flex card → buyer taps Approve/Decline → agent completes purchase
    • 
            
  • Checkout Receipt — After ACP complete, push a receipt card to the buyer's LINE with item, total, merchant, payment method
    • 
            
  • x402 Crypto Payments — When payment uses x402 (HTTP 402 flow), the confirmation card shows tx hash, network, token, and links to block explorer
    • 
            
  • Order Updates — Status changes (shipped, delivered, canceled) push real-time cards to LINE
    • 
          

          
Integration Architecture

          
    
            
  • acp-handler webhooks → notify-bridge → LINE Push API → Flex card to buyer
    • 
            
  • x402 payment confirmation → notify-bridge → LINE Flex with tx details
    • 
            
  • LINE webhook → notify-bridge → capture userId for future pushes
    • 
            
  • Future: LINE Login for user identity → map to ACP sessions
    •
